# Employee Dashboard

A frontend dashboard built with **React**, **Vite**, and **AG Grid** for displaying and managing employee data. Features interactive charts using **Recharts** for department and performance visualizations.

## Features

- AG Grid table with employee data (sorting, filtering)
- Department distribution chart
- Performance overview chart
- Summary cards with key metrics

## Tech Stack

- React 19
- Vite 7
- AG Grid (Community Edition)
- Recharts

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or above)

### Installation

```bash
# Clone the repository
git clone https://github.com/Megha-Mishra/employee-dashboard.git
cd employee-dashboard

# Install dependencies
npm install

# Start the development server
npm run dev
